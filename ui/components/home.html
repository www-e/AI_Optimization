Welcome to Hybrid AI Optimization

This application demonstrates the use of nature-inspired optimization algorithms to train neural networks without traditional gradient descent. The project includes four different algorithms:

Genetic Algorithm

Inspired by natural evolution, using selection, crossover, and mutation to evolve a population of solutions.

Particle Swarm Optimization

Inspired by social behavior of birds flocking or fish schooling, using particles that move through the search space.

Ant Colony Optimization

Inspired by ants finding paths through pheromone trails, using artificial ants to construct solutions.

Tabu Search

A metaheuristic that uses memory structures to avoid revisiting recently explored solutions.

Dataset: Iris Classification

This project uses the famous Iris dataset for classification. The neural network is trained to classify iris flowers into three species based on four features:

  • Sepal Length (cm)
  • Sepal Width (cm)
  • Petal Length (cm)
  • Petal Width (cm)

The three species to classify are:

  • Setosa
  • Versicolor
  • Virginica

Neural Network Architecture

The neural network used in this project is a simple feedforward network with one hidden layer:

Input Layer

4 neurons (one for each feature)

Hidden Layer

8 neurons with sigmoid activation

Output Layer

3 neurons (one for each class) with softmax activation

The optimization algorithms are used to find the optimal weights for this neural network, replacing traditional gradient descent methods like backpropagation.

Getting Started

To get started with the application:

  1. Select an algorithm from the sidebar to explore its parameters and run it individually.
  2. Use the "Compare All" option to run and compare all algorithms side by side.
  3. Adjust parameters for each algorithm to see how they affect performance.
  4. View the results including accuracy, execution time, and convergence charts.
